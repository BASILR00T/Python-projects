# Rock-Paper-Scissors Game

A simple command-line Rock-Paper-Scissors game where you play against the computer.

## Features
- Play Rock-Paper-Scissors against the computer.
- Track wins, losses, and ties over 3 rounds.
- Option to play again after 3 rounds.

## How to Use
1. Run the script `Game.py`.
2. Enter your choice (`rock`, `paper`, or `scissor`) when prompted.
3. The computer will randomly select its choice.
4. The result (win, lose, or tie) will be displayed.
5. After 3 rounds, the game will display your total wins, losses, and ties.
6. You can choose to play again or exit the game.
